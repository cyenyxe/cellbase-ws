Working with External References

With this application it is possible to search easily for any external reference of an identifier. External references are divided into different types:

  • Functional Annotation
  • Identifier
  • Microarray probeset
  • Pathway
  • Regulatory region

A list of the possible databases are available typing the following URLs:

DataBases
All http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/feature/dbnames
Identifier http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/feature/dbnames?type=Identifier
Microarray probeset 
http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/feature/dbnames?type=Microarray Probeset
Pathway http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/feature/dbnames?type=Pathway
Regulatory region http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/feature/dbnames?type=Regulatory Region

It is also possible to combine different database names:

http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/feature/dbnames?type=Pathway,Regulatory Region

Working with Genomic

Genomic category deals with the concept of genomic location in order to get specific information. That way, with this tool it is possible to retrieve biological data about a location. The concept "location" can be specified in different ways depending on what we are looking for:

  • Position: Chromosome and position. E.g.: 1:893801.
  • Region: Chromosome, start and end. It is also possible to introduce a whole chromosome indicating the number.
    E.g.: 5:958453-962854; 21.
  • Cytoband: Comming soon.

You can concatenate positions or regions (not mixed) typing them one after the other and separating them using a comma.
E.g.: 3:893801-1000000,5:958453-962854,13,4:156832030-358639550,8
E.g.: 1:845962,16:54854,11:38745,7:68786307
The available URLs provide the user with information about:

Genes

It is possible to know all genes contained in a region and also in a whole chromosome. This application is also provided with a Biotype filter where you can delimit the results for one ore more biotypes. The list with the possible biotypes is here. Such queries have the following format:

http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/genomic/region/{region}/gene?biotype={biotype}
Some examples:
http://bioinfo.cipf.es/infrared-ws/api/v1/hsa/genomic/region/2:1000-1200000,5:234234-4564564,21/gene http://bioinfo.cipf.es/infrared-ws/api/v1/hsa/genomic/region/5:234234-4564564,21?biotype=protein_coding,misc_RNA

SNPs (Single Nucleotide Polimorfisms)

SNP information can be accesed through a region or a position. If your query is related to one or more regions, it is possible also to filter by Consequence type. The list with the possible consequence types is here. If your query is related to some specific positions, the search will let you know if there is any SNP related. The two types of search look like this:

http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/genomic/region/{region}/snp?consequencetype={consequencetype}
http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/genomic/position/{position}/snp
Some examples:
http://bioinfo.cipf.es/infrared-ws/api/v1/hsa/genomic/region/2:1000-1200000,5:234234-4564564,21/snp
http://bioinfo.cipf.es/infrared-ws/api/v1/hsa/genomic/region/5:1-800000/snp?consequencetype=3PRIME_UTR,UPSTREAM
http://bioinfo.cipf.es/infrared-ws/api/v1/hsa/genomic/position/5:11784,10:60325/snp

Splice Sites

It is possible to know if one or more postions belong to a splice site. The way to access that information follow the same patron as explained above:

http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/genomic/position/{position}/splicesite
Example:
http://bioinfo.cipf.es/infrared-ws/api/v1/hsa/genomic/...

Conserved Regions

It is possible to know if one or more positions belong to a conserved region. The way to access that information is the following:

http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/genomic/position/{position}/conservedregion
Example:
http://bioinfo.cipf.es/infrared-ws/api/v1/hsa/genomic/position/12:67700,4:45000,6:48752/conservedregion

TFBS (Transcription Factor Binding Sites)

By entering one or more positions in the URL you will be able to know whether these particular locations belong to a transcription factor binding site. It is necessary to specify a filter by database and the available options are:

  • Jaspar
  • ORegAnno

The URL query will look like this:

http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/genomic/position/{position}/tfbs?filter={db}
Examples:
http://bioinfo.cipf.es/infrared-ws/api/v1/hsa/genomic/position/1:893801,9:785426,6:46096774/tfbs?filter=jaspar 
http://bioinfo.cipf.es/infrared-ws/api/v1/hsa/genomic/position/2:128175992,4:156832030,X:33229755/tfbs?filter=oreganno

miRNA Targets

In order to know whether one or more positions are targets of a miRNA you can run the following query:

http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/genomic/position/{position}/mirna_target
Example:
http://bioinfo.cipf.es/infrared-ws/api/v1/hsa/genomic/position/2:120155845,17:30743/mirna_target

miRNA Genes

It is possible to know if one or more positions belong to a miRNA gene. The way to access that information is the following:

http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/genomic/position/{position}/mirna_gene
Example:
http://bioinfo.cipf.es/infrared-ws/api/v1/hsa/genomic/position/2:120155845,17:30743/mirna_gene

Triplex

To know if one or more positions belong to a triplex. You can access typing a query in that way:

http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/genomic/position/{position}/triplex
Example:
http://bioinfo.cipf.es/infrared-ws/api/v1/hsa/genomic/position/2:120155845,17:30743/triplex

Variation

The aim of this implementation is to offer the user an easy way to retrieve any information related with genetic variation, such as general information, consequence type, omega codon values, etc. Results can be accessed just adding your reference IDs (rs) separated by commas in the query. The general URL looks like this:

http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/variation/{variation_IDs}/{feature}

Depending on the feature choosen, it is possible to filter the results by some extra features (i.e: consequence type, population).


SNP Information

This implementation lets the user easily know some general features about a collection of SNP IDs. Results will contain information about the location of the SNPs in the genome, the strand, the bases involved and the consequence type. Such queries will have the following format:

http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/variation/{variation_IDs}/info
Example:
http://bioinfo.cipf.es/infrared-ws/api/v1/hsa/genomic/region/2:1000-1200000,5:234234-4564564,21/gene

Consequence Type

Information about the consequence type/s of a collection of SNP IDs can be accessed through the following query:

http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/variation/{variation_IDs}/consequencetype This implementation lets the user easily know some general features about a collection of SNP IDs. Results will contain information about the location of the SNPs in the genome, the strand, the bases involved and the consequence type. Such queries will have the following format: http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/variation/{variation_IDs}/info
Example:
