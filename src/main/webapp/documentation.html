Working with External References

With this application it is possible to search easily for any external reference of an identifier. External references are divided into different types:

  • Functional Annotation
  • Identifier
  • Microarray probeset
  • Pathway
  • Regulatory region

A list of the possible databases are available typing the following URLs:

DataBases
All http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/feature/dbnames
Identifier http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/feature/dbnames?type=Identifier
Microarray probeset http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/feature/dbnames?type=Microarray Probeset
Pathway http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/feature/dbnames?type=Pathway
Regulatory region http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/feature/dbnames?type=Regulatory Region

It is also possible to combine different database names.
