Welcome to the Infrared WS!

Infrared is a complete set of Web APIs (Application programming Interfaces) based on REST (Representational State Transfer) architecture. The aim of this implementation is to provide an easy way of retrieving biological data from the Infrared DB. Infrared DB contains information on ... gathered from ....

The API allows the user a deep and efficient access to the database tables from a friendly interface based on the following structure:
http://bioinfo.cipf.es/infrared-ws/api/{version}/{specie}/{category}/{id}/{feature}?{other_features}

For example, to list all the SNPs contained in a particular genomic region like chomosome 3 from the position 12000 to 500000 in human, we just have to type:
http://bioinfo.cipf.es/infrared-ws/api/hsa/genomic/3:12000-500000/snp
